Proyectos

Predicción de precios automotores usando Selenium y Python

El proyecto trata sobre la extracción y análisis de datos de precios de venta de automotores según sus diferentes características, y el posterior uso de estos datos para crear un modelo para predecir los precios de un automotor dadas sus características.

Para ello, utilizando técnicas de “web scraping” obtenemos información de todos los autos publicados en mercado libre uruguay en un momento dado, y luego, utilizamos diferentes herramientas estadísticas en python para crear un modelo y obtener predicciones con un promedio de error por debajo del 15% en un proyecto que solo tomo un fin de semana.
